# Java_PrintEE
Sick of having to type `System.out.println` to print something in Java? Now you can just type `print`!
## How to use it
Example:

myProgram.java
```console
public class myProgram {
  static void print(String printEE) {
    System.out.println(printEE);
  }
  print("Hello, World!");
}
```
